## 📋 Skill Assessment Plan

### Phase 1: Technical Readiness (Week 0)
**Goal**: Understand baseline familiarity with tools and codebase

**Questions to Ask:**
1. **OED Familiarity**
   - Have you explored the OED dashboard locally?
   - Do you understand what the application does?
   - Have you tested the different chart types?

2. **Development Environment**
   - Can you run the application locally?
   - Is Docker working for you?
   - Can you run the test suite successfully?

3. **GitHub Experience**
   - How comfortable are you with Git/GitHub?
   - Have you done pull requests before?
   - Familiar with branching strategies?

4. **Testing Knowledge**
   - Have you written automated tests before?
   - Familiar with Mocha/Chai or similar frameworks?
   - Understand the purpose of the tests we're writing?

### Phase 2: Task Readiness (Week 1)
**Goal**: Ensure everyone can start working independently

**Check for Understanding:**
- Can explain what their assigned test validates
- Knows where to find expected CSV files
- Understands test structure from examples
- Can identify blockers early
