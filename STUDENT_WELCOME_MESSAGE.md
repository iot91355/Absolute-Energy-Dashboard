## ⚠️ Key Things to Know

### 1. **Do NOT Fork This Repo**
❌ Wrong: Fork OED-dungo to your account  
✅ Right: Clone OED-dungo directly and work on branches

**Why?** We're simulating a real team - everyone works in the same repo via branches!

```bash
# Do this:
git clone https://github.com/aldungo/OED-dungo.git

# NOT this:
# (Click fork button on GitHub)
```

### 2. **Create Feature Branches**
```bash
git checkout -b feature/test-L4-adam
# Format: feature/test-[ID]-[yourname]
```

### 3. **Open Pull Requests**
- Create PR from your branch → `development`
- Request review from 1-2 teammates
- We'll review and merge together

### 4. **Daily Standups**
Post in Discord `#standups` each day:
```
📅 Standup - [Your Name] - [Date]

✅ Yesterday: [what you did]
🔨 Today: [what you're working on]
🚧 Blockers: [any issues or "None"]
⏱️ Hours: [time worked]
```
